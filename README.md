# ocean-spray-pc

## 后台管理系统

**开发前** 请开发人员查看项目代码规范 [docs/项目代码规范.md](./docs/项目代码规范.md)

### 框架

```js
Vite3 + Vue2 + ElementUi;
```

## 项目启动

```js
yarn // 安装依赖

yanr serve // 运行开发环境
yarn serve.pre // 运行业务演示环境
yarn serve.prod // 运行正式环境

yanr build // 打包开发环境
yarn build.pre // 打包业务演示环境
yarn build.prod // 打包正式环境
```
